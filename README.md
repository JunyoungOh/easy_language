# easy_language — 쉬운 말로 설명해주는 Claude 스킬

비개발자가 Claude Code와 나누는 대화를, 전문용어를 풀어주고 일상 비유를 곁들인
쉬운 한국어로 바꿔주는 스킬입니다.

## 이게 뭔가요?

Claude Code는 기본적으로 개발자에게 맞춘 말투로 답합니다 — "배포",
"리팩터링", "API" 같은 전문용어가 자주 나오죠. 이 스킬을 켜면 Claude가:

- 전문용어를 처음 쓸 때 괄호나 비유로 풀어줍니다
- 추상적인 개념을 일상 사물에 빗대 설명합니다
- 짧고 명확한 문장으로, 결론과 이유를 먼저 말합니다
- 코드는 그대로 보여주되 "무엇을, 왜" 한 줄 설명을 붙입니다

코드 파일·커밋 메시지·주석 같은 산출물은 건드리지 않습니다. 바뀌는 건
오직 Claude가 사용자에게 말을 거는 대화 텍스트뿐입니다.

## 설치

이 저장소를 Claude Code가 스킬을 읽어오는 폴더에 복제합니다:

```bash
git clone https://github.com/JunyoungOh/easy_language.git ~/.claude/skills/plain-language
```

새 세션(새 대화)을 시작하면 스킬이 인식됩니다.

## 사용법

**직접 호출** — 대화 중 아무 때나:

```
/plain-language
```

**1회 설정 (계속 적용)** — "나는 비개발자야, 앞으로 쉽게 설명해줘"처럼
말하면 스킬이 설정 모드로 들어갑니다. 적용 범위(이 폴더만 / 모든 작업)를
물은 뒤 `CLAUDE.md`에 스위치 한 줄을 적어, 다음 세션에도 유지되게 합니다.

**해제** — "쉬운 말 그만"이라고 말하면 그 스위치를 제거합니다.

## 작동 방식

스킬의 모든 규칙은 `SKILL.md` 한 파일에 들어 있습니다. `CLAUDE.md`에
적히는 것은 스킬을 켜는 짧은 스위치 한 줄뿐이라, 규칙을 고칠 때는
`SKILL.md`만 수정하면 됩니다.

## 만든 경위

Claude Code의 skill-creator로 제작했습니다.
